---

## DASAR HUKUM

Peraturan Menteri Pendayagunaan Aparatur Negara dan Reformasi Birokrasi Republik Indonesia Nomor 14 Tahun 2017 tentang Pedoman Penyusunan Survei Kepuasan Masyarakat Unit Penyelenggara Pelayanan Publik.

---

## KONSEP DASAR

IKM adalah data dan informasi tentang tingkat kepuasan masyarakat yang diperoleh dari hasil pengukuran secara kuantitatif dan kualitatif atas pendapat masyarakat dalam memperoleh pelayanan dari aparatur penyelenggara pelayanan publik dengan membandingkan antara harapan dan kebutuhannya.

## RUMUS PERHITUNGAN IKM

### LANGKAH 1: Menghitung Nilai Rata-rata per Unsur (NRR per Unsur)

```
         Σ Nilai per Unsur
NRR = ─────────────────────────
        Jumlah Responden
```

**Keterangan:**
- NRR = Nilai Rata-rata per unsur
- Σ Nilai per Unsur = Total nilai yang diberikan semua responden untuk satu unsur
- Jumlah Responden = Total responden yang mengisi kuesioner

**Contoh Perhitungan U1 (Persyaratan Pelayanan):**

Data responden untuk U1:
- Responden 1-18 memberi nilai 4 (18 orang)
- Responden 19-29 memberi nilai 3 (11 orang)

```
Total Nilai U1 = (18 × 4) + (11 × 3)
               = 72 + 33
               = 105

NRR U1 = 105 / 29
       = 3.6207
```

---

### LANGKAH 2: Menghitung Bobot per Unsur

```
Bobot = 1 / Jumlah Unsur Pelayanan
```

Karena ada 9 unsur pelayanan:

```
Bobot = 1 / 9 = 0.1111 (atau 11.11%)
```

**Catatan:** Bobot sama untuk semua unsur (bobot seimbang).

---

### LANGKAH 3: Menghitung NRR Tertimbang per Unsur

```
NRR Tertimbang = NRR per Unsur × Bobot
```

**Contoh Perhitungan untuk U1:**

```
NRR Tertimbang U1 = 3.6207 × 0.1111
                  = 0.3983
```

**Atau dalam persen:**

```
NRR Tertimbang U1 (%) = 3.6207 × 11.11%
                      = 40.23%
```

---

### LANGKAH 4: Menghitung Total NRR Tertimbang

```
Total NRR Tertimbang = Σ (NRR Tertimbang semua unsur)
                     = NRR Tertimbang U1 + U2 + ... + U9
```

**Perhitungan Lengkap:**

```
Total NRR Tertimbang = 0.3983 + 0.3831 + 0.3793 + 0.4400 + 
                       0.3869 + 0.3983 + 0.4021 + 0.3983 + 
                       0.4362
                     = 3.6224
```

**Atau cara alternatif (rata-rata dari NRR semua unsur):**

```
Total NRR Tertimbang = (3.6207 + 3.4828 + 3.4483 + 4.0000 + 
                        3.5172 + 3.6207 + 3.6552 + 3.6207 + 
                        3.9655) / 9
                     = 32.9311 / 9
                     = 3.6590
```

---

### LANGKAH 5: Menghitung Nilai IKM Unit Pelayanan

```
IKM = Total NRR Tertimbang × 25
```

atau

```
IKM = (Σ NRR semua unsur / Jumlah Unsur) × 25
```

**Perhitungan:**

```
IKM = 3.6224 × 25
    = 90.56
```

**Penjelasan angka 25:**
- Skala penilaian 1-4 dikonversi ke skala 25-100
- Nilai minimum: 1 × 25 = 25
- Nilai maksimum: 4 × 25 = 100
- Interval: (100-25)/4 = 18.75

## RUMUS DALAM SATU FORMULA

### Formula Komprehensif:

```
         ⎡  n   9                    ⎤
         ⎢  Σ   Σ  Nilai(i,j)        ⎥
         ⎢ i=1 j=1                   ⎥
IKM = 25 × ⎢─────────────────────── ⎥
         ⎢      n × 9                ⎥
         ⎣                           ⎦
```

**Dimana:**
- n = Jumlah responden (29)
- 9 = Jumlah unsur pelayanan
- i = Responden ke-i
- j = Unsur pelayanan ke-j
- Nilai(i,j) = Nilai yang diberikan responden i untuk unsur j
- 25 = Konstanta konversi skala

### Formula Sederhana:

```
IKM = (Total Semua Nilai / (Jumlah Responden × Jumlah Unsur)) × 25
```

**Perhitungan Praktis:**

```
IKM = (955 / (29 × 9)) × 25
    = (955 / 261) × 25
    = 3.6590 × 25
    = 91.48
```

**Catatan:** Ada sedikit perbedaan karena pembulatan di langkah-langkah sebelumnya.

## KESALAHAN UMUM YANG HARUS DIHINDARI

❌ **1. Salah Bobot**
```
SALAH: Bobot = 1/10 = 0.1 (harusnya 1/9)
```

❌ **2. Lupa Kalikan 25**
```
SALAH: IKM = 3.6590 (harusnya × 25)
BENAR: IKM = 3.6590 × 25 = 91.48
```

❌ **3. Pembulatan Terlalu Dini**
```
SALAH: NRR = 3.6 (pembulatan terlalu dini)
BENAR: NRR = 3.6207 (gunakan minimal 4 desimal)
```

❌ **4. Salah Hitung Total Nilai**
```
SALAH: Hanya menghitung responden nilai 4
BENAR: Hitung semua responden (nilai 1-4)
```

❌ **5. Salah Kategori**
```
SALAH: IKM 90.56 → Kategori B
BENAR: IKM 90.56 → Kategori A (≥88.31)
```
